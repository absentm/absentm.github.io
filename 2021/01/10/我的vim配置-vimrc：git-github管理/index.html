¶使用Bundle命令管理 vimrc

vim 相关的所有插件都存放在 bundles.vim中，可以自己向其中添加喜爱的插件，

使用命令如下：

1: :BundleClean 清理所有失效或无用的插件
2: :BundleInstall 安装新添加的vim插件
3: :BundleInstall! 更新所有插件
